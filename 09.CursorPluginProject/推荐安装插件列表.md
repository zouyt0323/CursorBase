# Cursor 推荐安装插件列表

> 根据当前工程习惯（Android/Java/Kotlin、Python、Figma、Docker、MCP 等）整理

---

## 一、Cursor 官方 Marketplace 插件（需在 Cursor 内安装）

在 Cursor 中：**设置 → Plugins → Browse Marketplace** 或使用 `/add-plugin` 命令。

### 设计 & 协作（与现有 Figma MCP 互补）

| 插件 | 说明 | 推荐理由 |
|------|------|----------|
| **Figma** | 设计转代码、Code Connect | 你已有 Figma MCP，官方插件可提供设计系统规则与一键实现 |
| **Linear** | 工单、项目、文档 | 若用 Linear 做项目管理，可在 Cursor 内直接访问 |

### 部署 & 基础设施

| 插件 | 说明 | 推荐理由 |
|------|------|----------|
| **Vercel** | 一键部署 React/Next.js | 若有 Web 前端项目，内置 Vercel 最佳实践 |
| **AWS** | 部署与管理 AWS 资源 | 若用 AWS 部署 |
| **Cloudflare** | 边缘部署与 CDN | 若用 Cloudflare |

### 支付 & 数据

| 插件 | 说明 | 推荐理由 |
|------|------|----------|
| **Stripe** | 支付集成、订阅产品 | 若做含支付的产品，可快速生成支付链接与订阅 |
| **Supabase** | 云数据库自然语言操作 | 若用 Supabase，可用自然语言操作数据库 |
| **Databricks** / **Snowflake** | 数据查询与分析 | 若做数据分析或数据工程 |

### 产品与分析

| 插件 | 说明 | 推荐理由 |
|------|------|----------|
| **Amplitude** | 行为分析、增长仪表盘 | 若做产品分析 |
| **Hex** | 数据笔记本与洞察 | 若做数据探索 |

## 二、VSCode 扩展市场（cursor --install-extension）

以下为通用开发增强，可通过 `cursor --install-extension <id>` 安装。

### Kotlin / Android（你已有 Java，可补充）

| 插件 ID | 说明 |
|---------|------|
| `mathiasfrohlich.kotlin` | Kotlin 语言支持 |
| `fwcd.kotlin` | Kotlin 语言服务（二选一） |

### 代码质量

| 插件 ID | 说明 |
|---------|------|
| `dbaeumer.vscode-eslint` | ESLint 代码检查 |
| `esbenp.prettier-vscode` | Prettier 格式化 |
| `usernamehw.errorlens` | 行内错误提示增强 |

### Git 增强

| 插件 ID | 说明 |
|---------|------|
| `eamodio.gitlens` | Git  blame、历史、对比增强 |

### 其他实用

| 插件 ID | 说明 |
|---------|------|
| `gruntfuggly.todo-tree` | TODO/FIXME 高亮与树形展示 |
| `aaron-bond.better-comments` | 注释高亮分类 |
| `streetsidesoftware.code-spell-checker` | 拼写检查 |

## 三、按需安装建议

| 场景 | 建议安装 |
|------|----------|
| 做 Web 前端 + 部署 | Figma、Vercel |
| 做支付/订阅产品 | Stripe |
| 用 Supabase 后端 | Supabase |
| 用 Linear 管理任务 | Linear |
| 提升 Git 体验 | GitLens |
| Kotlin 开发 | Kotlin 插件 |
| 代码规范 | ESLint、Prettier |

## 四、安装命令速查

```bash
# Cursor Marketplace 插件（Figma、Stripe、Vercel 等）需在 Cursor 内通过 设置→Plugins→Browse 或 /add-plugin 安装

# 以下 VSCode 扩展已通过 CLI 安装（2026-03-18）
cursor --install-extension mathiasfrohlich.kotlin        # ✓ 已安装
cursor --install-extension dbaeumer.vscode-eslint        # ✓ 已安装
cursor --install-extension esbenp.prettier-vscode        # ✓ 已安装
cursor --install-extension eamodio.gitlens               # ✓ 已安装
cursor --install-extension usernamehw.errorlens          # ✓ 已安装
cursor --install-extension gruntfuggly.todo-tree         # ✓ 已安装
cursor --install-extension aaron-bond.better-comments     # ✓ 已安装
cursor --install-extension streetsidesoftware.code-spell-checker  # ✓ 已安装
```
