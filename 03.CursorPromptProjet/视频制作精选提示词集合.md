## 三、AI 文生视频提示词

### 3.1 Sora 2 电影级提示词模板

**出处：** [OpenAI Cookbook - Sora 2 Prompting Guide](https://developers.openai.com/cookbook/examples/sora/sora2_prompting_guide/) + [SkyWork - Best Practices for Sora 2](https://skywork.ai/blog/sora-2-prompt-authoring-best-practices-2025/) + [soprompts.com - 50+ Best Sora 2 Prompts](https://soprompts.com/blog/best-sora-2-prompts)（免费）

**通用模板结构：**
```
[Scene Setup / 场景设定] + [Subject & Action / 主体与动作] + 
[Camera Grammar / 镜头语言] + [Lighting & Color / 光影与色彩] + 
[Physics & Materials / 物理质感] + [Audio Cues / 音效提示]
```

**示例 1：史诗级风景**
```
Sweeping aerial view over misty mountain peaks at sunrise, camera slowly 
descending through clouds. Golden sunlight breaking through mist, revealing 
a pristine valley with winding river. Cinematic style with smooth drone 
movement, warm color grading, volumetric lighting, 4K quality.
```

**示例 2：录音棚氛围**
```
Warm, nostalgic recording studio interior; handheld 35mm feel with shallow 
depth of field and soft tungsten glow. Slow dolly push toward a woman 
singing into a vintage microphone. Palette: brass, mahogany, soft rose, 
coal gray. Ambient audio: quiet tape hiss, monitor hum.
```

**示例 3：奇幻场景**
```
Wide aerial of medieval fortress in morning mist; slow orbiting camera 
motion. Dragon glides overhead, wings catching sunset light as fog ripples 
below. Palette: gold, obsidian, cloud gray, moss green. Audio: rushing 
wind, deep wingbeat, distant horns.
```

**示例 4：产品广告**
```
Macro close-up of premium watch on dark marble surface. Camera slowly 
orbiting, revealing intricate dial details. Warm studio lighting with 
subtle caustic reflections. Shallow depth of field, bokeh background. 
Smooth 360-degree rotation. Audio: quiet mechanical ticking.
```

**Sora 2 提示词关键技巧：**
- 用具体描述代替抽象词（"slow tracking shot" 而非 "cinematic"）
- 加入感官细节（"steam from the oven fogging the window"）
- 利用原生音频同步（"faint jazz from the radio"）
- 每个镜头保持一个清晰的摄像机运动 + 一个决定性动作
- 迭代优化：生成 → 审查 → 调整 → 重新生成

### 3.2 Runway Gen-4 提示词模板

**出处：** [Runway Help Center - Gen-4 Video Prompting Guide](https://help.runwayml.com/hc/en-us/articles/39789879462419-Gen-4-Video-Prompting-Guide) + [FocalML - Exploring Runway Gen-4](https://focalml.com/blog/exploring-runway-gen-4-tips-for-crafting-professional-grade-videos/)（免费）

**核心公式：**
```
[Subject / 主体] + [Action / 动作] + [Environment / 环境] + 
[Lighting & Time / 光影与时间] + [Lens & Camera Move / 镜头运动] + 
[Mood & Texture / 情绪与质感]
```

**示例 1：运动慢镜头**
```
A slow-motion shot of a skateboarder doing a kickflip at golden hour, 
dust kicking up from the pavement, soft depth of field, handheld camera, 
cinematic film grain.
```

**示例 2：人物肖像**
```
A young woman looking directly into camera, gentle smile forming on her 
face, soft window light illuminating her features from the left, locked 
camera, shallow depth of field, warm tones with film grain texture.
```

**示例 3：城市夜景**
```
Neon-lit city street at night after rain, camera slowly tracking forward 
through puddle reflections, bokeh lights in background, steam rising from 
manhole, moody blue-pink color palette, anamorphic lens flare.
```

**Runway Gen-4 关键原则：**
- 拥抱简洁：从简单提示开始，逐步添加细节
- 只用正向描述：描述"应该发生什么"，不要写"不要xxx"
- 聚焦运动：图片提供视觉信息，文字重点描述动态
- 每个提示词一个核心视觉创意
- 输出时长：5 或 10 秒

### 3.3 可灵（Kling）文生视频提示词模板

**出处：** [AI学习与分享 - 可灵AI文生视频基础教程](https://www.aiools.com/1468.html) + [腾讯云 - AI视频制作全攻略](https://cloud.tencent.cn/developer/article/2586530)（免费）

**核心公式：**
```
主体（主体描述）+ 运动 + 场景（场景描述）+（镜头语言 + 光影 + 氛围）
```

**示例 1：美食特写**
```
一杯冒着热气的拿铁咖啡，缓缓倒入牛奶形成漂亮的拉花图案，
放在木质桌面上，旁边有一本打开的书，阳光从窗户洒入，
特写镜头，浅景深，暖色调，温馨惬意的氛围。
```

**示例 2：自然风光**
```
一只白色小猫在樱花树下嬉戏，花瓣随风飘落，
绿草地上阳光斑驳，背景虚化的粉色花海，
中景跟拍，柔光，电影级调色，治愈系氛围。
```

**示例 3：科技产品**
```
一部银色智能手机悬浮在深空蓝背景中，缓慢旋转展示机身侧面，
光线从顶部打下形成高光和阴影，金属质感细腻可见，
特写镜头缓慢环绕，科技感光效，极简主义风格。
```

**示例 4：人物叙事**
```
一位穿着白色连衣裙的女孩站在海边，迎风微笑，
长发随风飘动，夕阳余晖映照在脸上形成金色光晕，
中景 slowly zoom in，逆光拍摄，电影感调色，
浪漫温暖的氛围。
```

**可灵提示词技巧：**
- 动作描述简洁，适合 5-10 秒展示
- 避免负面描述（"不要模糊"会被误读）
- 画面保持简单，不要堆叠过多细节
- 复杂场景用图生视频代替纯文生视频
- 先用文生图预审构图再生成视频

## 四、视频后期制作

### 4.1 视频剪辑节奏指南（Sigma Edit 风格）

**出处：** [DocsBot - Sigma Edit Video Guide](https://docsbot.ai/prompts/creative/sigma-edit-video-guide)（免费）

```
Create a detailed guide for making a dynamic edit video synchronized 
to music. Provide step-by-step instructions:

# Steps

1. **Select Audio Track / 选择音轨**
   - Choose a track with clear beats and rhythm changes
   - Identify BPM and key transition points

2. **Gather Video Clips / 素材收集**
   - Organize clips by theme, energy level, and visual style
   - Tag clips with metadata (duration, action type, emotion)

3. **Import & Analyze / 导入分析**
   - Import clips and audio into editing software
   - Create beat markers on the timeline

4. **Beat Sync Editing / 卡点剪辑**
   - Cut clips precisely on beats and rhythm changes
   - Match visual transitions to audio transitions
   - Use speed ramping (slow-mo ↔ fast motion) for impact

5. **Transitions & Effects / 转场与特效**
   - Zoom transitions on bass drops
   - Whip pans on snare hits
   - Flash/glow effects on melodic peaks
   - Shake effect on impact sounds

6. **Color Grading / 调色**
   - Apply consistent color grade across clips
   - Use color shifts to match emotional arcs of the music
   - Cinematic LUT suggestions for the desired mood

7. **Speed Ramping / 变速**
   - Slow motion before beat drops for anticipation
   - Speed up during high-energy sections
   - Smooth speed transitions

8. **Final Polish / 最终润色**
   - Review sync accuracy frame by frame
   - Adjust audio levels and add sound effects
   - Export in optimal quality for target platform

推荐剪辑工具：DaVinci Resolve / Premiere Pro / 剪映 / CapCut
```

## 六、完整视频制作工作流

### 6.1 从零到成片的 AI 视频制作流程

```

【第一阶段：Pre-Production / 前期策划】

Step 1: 创意策划
- 使用 ChatGPT/Claude 头脑风暴视频创意
- 确定目标受众和平台
- 分析同类爆款视频的关键元素

Step 2: 脚本编写
提示词：
"请为 [主题] 写一个 [时长] 的短视频脚本，包含：
旁白文案、画面描述、BGM 建议、转场指示。
风格：[风格]，目标平台：[平台]"

Step 3: 分镜设计
提示词：
"将以上脚本转化为详细分镜表，每个镜头包含：
镜号 | 时长 | 景别 | 运镜 | 画面描述 | 对应AI生成提示词"

【第二阶段：Production / 素材生成】

Step 4: AI 文生图（预审构图）
- 工具：Midjourney / DALL-E / Flux
- 为每个镜头生成静态参考图
- 确认构图、色调、风格一致性

Step 5: AI 文生视频 / 图生视频
- 工具：Sora 2 / Runway Gen-4 / 可灵 / 即梦
- 每个镜头生成 3-5 个候选版本
- 选择最佳版本作为最终素材

Step 6: AI 配音 / 配乐
- 旁白：ElevenLabs / 通义听悟 / 剪映 TTS
- 配乐：Suno / Udio / Epidemic Sound
- 音效：Freesound / Artlist

【第三阶段：Post-Production / 后期制作】

Step 7: 剪辑组装
- 工具：DaVinci Resolve / Premiere Pro / 剪映
- 按分镜表排列素材
- 添加转场和速度调整

Step 8: 调色与特效
- 统一全片色调
- 添加必要特效
- 字幕排版

Step 9: 终审导出
- 多平台适配（横版/竖版/方形）
- 画质优化
- 封面和标题优化

【效率参考】
传统流程：3-5 天 → AI 工作流：3-5 小时（效率提升 70%+）
```
