## 使用建议

- **想用子代理 / 拆任务**：先看 [Subagents 文档](https://cursor.com/docs/context/subagents) 与 [Changelog](https://cursor.com/changelog)。
- **想扩展 Agent 能力**：用 [Skills 文档](https://cursor.com/docs/context/skills) + [Agent Skills Hub](https://www.cursorhow.com/en/agent-skills-hub) + [cursor-skills](https://github.com/daniel-scrivner/cursor-skills)。
- **找 MCP 服务器**：优先 [官方 MCP 目录](https://cursor.com/docs/context/mcp/directory)、[cursormcp.dev](https://cursormcp.dev)、[cursor.directory/mcp](https://cursor.directory/mcp)。
- **配项目规则**：用 [cursorrules.org](https://www.cursorrules.org/zh) 或 [prpm.dev](https://prpm.dev) 生成/管理 .cursorrules。
- **中文学习路径**：[learn-cursor.com](https://learn-cursor.com) + [cursor-cn.org](https://www.cursor-cn.org/docs) + [53ai.com 子代理报道](https://www.53ai.com/news/LargeLanguageModel/2026012303645.html)。

如有新站或分类建议，可在此文档基础上增补。
