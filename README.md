# CursorBase

Cursor 工程基座：规则、提示词、Skills 文档与 MCP 等资源汇总。

## 目录结构

| 目录 | 说明 |
|------|------|
| `01.CursorBaseProject` | Cursor 基础项目模板（规则、技能、Agent 等） |
| `02.CursorRuleProject` | Cursor Rules 规则与分类整理 |
| `03.CursorPromptProjet` | 各类场景提示词集合 |
| `04.CursorSkillProject` | Skills 文档与脚本 |
| `05.CursorSubAgentProject` | SubAgent 相关资源 |
| `06.CursorMCPProject` | MCP 相关资源 |
