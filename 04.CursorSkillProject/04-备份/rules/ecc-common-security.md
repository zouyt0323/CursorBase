# 安全指南

## 提交前必检项

在任意提交前确认：
- [ ] 无硬编码密钥（API 密钥、密码、令牌）
- [ ] 所有用户输入已验证
- [ ] 防止 SQL 注入（参数化查询）
- [ ] 防止 XSS（HTML 转义/过滤）
- [ ] 已启用 CSRF 保护
- [ ] 认证/授权已验证
- [ ] 所有端点有速率限制
- [ ] 错误信息不泄露敏感数据

## 密钥管理

- 绝不在源码中硬编码密钥
- 始终使用环境变量或密钥管理服务
- 启动时校验必需密钥是否存在
- 若密钥可能泄露，立即轮换

## 安全响应流程

发现安全问题后：
1. 立即停止
2. 修复严重问题后再继续
3. 轮换已泄露的密钥
4. 检查整个代码库中类似问题
