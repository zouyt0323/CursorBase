## Chrome DevTools MCP 配置

```json
{
  "chrome-devtools": {
    "command": "npx",
    "args": ["-y", "chrome-devtools-mcp@latest"]
  }
}
```

**功能：** 检查实时 DOM、捕获网络追踪、运行 Lighthouse 审计、截图、调试可访问性、分析性能指标。

**前置条件：** Node.js v20+、Chrome 浏览器、npm
