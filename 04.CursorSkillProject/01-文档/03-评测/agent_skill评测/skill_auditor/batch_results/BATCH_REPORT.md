## 改进建议

根据批量评测结果，以下是提升 Skill 质量的优先级建议：

1. **添加排除条件**（92% 缺失）：在 Description 中明确标注何时不应使用该 Skill
2. **使用 ASCII 决策树**（80% 缺失）：用树状结构替代冗长的 IF-ELSE 文字描述
3. **添加正反对比示例**（71% 缺失）：同时展示错误写法和正确写法
4. **包含可执行代码示例**（61% 缺失）：代码示例的沟通效率远高于纯文字解释
5. **提升 Description 动作描述**（50% 缺失）：在 Description 中明确声明「做什么」
