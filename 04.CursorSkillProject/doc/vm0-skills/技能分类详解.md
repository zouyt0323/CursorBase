# vm0-skills 技能分类详解

## 概述

vm0-skills 是一个遵循 [Agent Skills 规范](https://agentskills.io/specification) 的 SaaS API 技能合集。

核心理念：**No MCPs. No SDKs. No CLI apps. Just curl.** — 让 AI Agent 仅通过 curl 命令就能与各种 SaaS 服务交互。

---

## 一、设计与创意（1 个）

### figma — Figma REST API

- **功能**：通过 REST API 访问和管理 Figma 设计文件
- **密钥需求**：`FIGMA_API_TOKEN`
- **核心能力**：
  - 读取设计文件、提取组件和样式
  - 导出图片（PNG/JPG/SVG/PDF）
  - 管理评论（读取/发布/删除）
  - 获取文件版本历史
  - 团队/项目/成员管理
  - 获取设计令牌和组件集
- **API 数量**：18 个端点
- **限制**：只读（无法修改设计），60 请求/分钟

## Cursor 中使用建议

### 推荐安装的高价值技能

| 优先级 | 技能 | 理由 |
|--------|------|------|
| 高 | figma | 设计开发协作 |
| 高 | github | 代码管理 |
| 高 | supabase | 后端即服务 |
| 高 | notion | 文档协作 |
| 中 | slack | 团队通信 |
| 中 | openai | AI 集成 |
| 中 | firecrawl | 数据采集 |
| 中 | sentry | 错误追踪 |
| 低 | cloudinary | 媒体管理 |
| 低 | resend | 邮件发送 |

### 安装方式

```bash
# 批量安装到 Cursor
SKILLS_DIR="$HOME/.cursor/skills"
for skill in figma github supabase notion slack; do
  mkdir -p "$SKILLS_DIR/$skill"
  cp skills/vm0-skills/$skill/SKILL.md "$SKILLS_DIR/$skill/"
done
```
