# UI UX Pro Max — 功能详解

## 一、技能概述

**UI UX Pro Max** 是一个 AI 设计智能技能，为构建专业 UI/UX 提供全方位的设计知识库和智能推荐引擎。

**中文名**：UI/UX 专业设计大师

**核心能力**：
- 根据产品类型自动生成完整的设计系统（配色、字体、布局、组件）
- 100 条行业特定推理规则，智能匹配最佳设计方案
- 67 种 UI 风格可选，96 套配色方案，57 组字体搭配
- 支持 13 个技术栈的特定最佳实践
- 内置 Pre-delivery 检查清单，防止常见 UI/UX 错误

## 二、旗舰功能：设计系统生成器（v2.0）

### 工作流程

```
用户请求 → 5 路并行搜索 → 推理引擎匹配 → 输出完整设计系统
```

1. **用户请求**：自然语言描述需求（如"为我的 SaaS 产品构建着陆页"）
2. **多域搜索**（5 路并行）：
   - 产品类型匹配（100 个类别）
   - 风格推荐（67 种风格）
   - 配色选择（96 套方案）
   - 着陆页模式（24 种模式）
   - 字体搭配（57 组组合）
3. **推理引擎**：
   - 匹配产品 → UI 类别规则
   - BM25 排名确定风格优先级
   - 过滤行业反模式
   - 处理 JSON 条件决策规则
4. **输出**：完整的设计系统
   - 布局模式 + 风格 + 配色 + 字体 + 特效
   - 应避免的反模式
   - Pre-delivery 检查清单

### 使用命令

```bash
# 生成设计系统（ASCII 输出）
python3 .cursor/skills/ui-ux-pro-max/scripts/search.py "beauty spa wellness" --design-system -p "Serenity Spa"

# 生成设计系统（Markdown 输出）
python3 .cursor/skills/ui-ux-pro-max/scripts/search.py "fintech banking" --design-system -f markdown

# 持久化设计系统到文件
python3 .cursor/skills/ui-ux-pro-max/scripts/search.py "SaaS dashboard" --design-system --persist -p "MyApp"
```

## 四、100 条行业推理规则

推理引擎为以下行业提供专属设计建议：

| 行业大类 | 细分举例 |
|----------|----------|
| **科技与 SaaS** | SaaS、Micro SaaS、B2B 企业、开发者工具、AI/聊天机器人平台 |
| **金融** | 金融科技、银行、加密货币、保险、交易仪表板 |
| **医疗** | 诊所、药房、牙科、兽医、心理健康 |
| **电商** | 通用电商、奢侈品、市场平台、订阅盒子 |
| **服务** | 美容/SPA、餐厅、酒店、法律、咨询 |
| **创意** | 作品集、代理商、摄影、游戏、音乐流媒体 |
| **新兴技术** | Web3/NFT、空间计算、量子计算、自动驾驶系统 |

每条规则包含：
- **推荐布局模式** — 页面结构建议
- **风格优先级** — 最匹配的 UI 风格
- **色彩情绪** — 行业适配色调
- **字体风格** — 字体人格匹配
- **关键特效** — 动画和交互建议
- **反模式** — 应避免的做法（如银行类不用"AI 紫/粉渐变"）

## 六、99 条 UX 指南（按优先级排列）

| 优先级 | 类别 | 影响级别 | 规则示例 |
|:------:|------|:--------:|----------|
| 1 | 无障碍 | **关键** | 色彩对比度 >= 4.5:1、焦点环可见、alt 文本、aria 标签、键盘导航 |
| 2 | 触摸与交互 | **关键** | 触摸目标 >= 44x44px、加载态按钮禁用、错误提示靠近问题处、cursor-pointer |
| 3 | 性能 | **高** | WebP + srcset + 懒加载、prefers-reduced-motion、内容跳动预留空间 |
| 4 | 布局与响应式 | **高** | viewport meta、最小字号 16px、禁止横向滚动、z-index 管理 |
| 5 | 字体与色彩 | **中** | 行高 1.5-1.75、行宽 65-75 字符、标题/正文字体人格匹配 |
| 6 | 动画 | **中** | 时长 150-300ms、用 transform/opacity、骨架屏加载 |
| 7 | 风格选择 | **中** | 风格匹配产品类型、全站一致性、SVG 图标替代 emoji |
| 8 | 图表与数据 | **低** | 按数据特征选择图表类型 |

## 十、在 Cursor 中的使用方法

### 安装

```bash
# 推荐方式：CLI 安装
npm install -g uipro-cli
cd /your/project
uipro init --ai cursor
```

### 使用

安装后，在 Cursor 的对话中直接用自然语言描述需求即可：

```
为我的 SaaS 产品构建一个着陆页
```

技能会自动激活并：
1. 分析你的需求
2. 生成完整设计系统（配色/字体/布局）
3. 根据技术栈生成代码
4. 执行 Pre-delivery 检查

### 高级用法：设计系统持久化

```bash
# 生成并持久化主设计系统
python3 .cursor/skills/ui-ux-pro-max/scripts/search.py "SaaS dashboard" --design-system --persist -p "MyApp"

# 创建页面级覆盖
python3 .cursor/skills/ui-ux-pro-max/scripts/search.py "checkout flow" --design-system --persist -p "MyApp" --page "checkout"
```

这会创建分层设计系统：
```
design-system/
├── MASTER.md           # 全局设计规范（颜色、字体、间距、组件）
└── pages/
    └── checkout.md     # 页面级覆盖（仅记录与 Master 的差异）
```
