# Cursor Rules 项目

> Rules 的索引、自研规则、第三方规则与工具的归类整理。

## 目录结构

| 目录 | 说明 |
|------|------|
| `00-索引与清单/` | Top 50 分类清单、收集网站速查 |
| `01-自研规则/` | 自研规则（01-08 分类、示例、本地化指南） |
| `02-第三方规则/` | awesome-cursorrules 等第三方规则库 |
| `03-工具/` | JNI 等辅助脚本 |

## 使用方式

1. **按分类选规则**：打开 `01-自研规则/` 下 `01-`～`08-` 对应文件
2. **示例规则**：`01-自研规则/examples/` 可复制到 `.cursor/rules/`
3. **第三方规则**：`02-第三方规则/awesome-cursorrules/` 含 rules、rules-new 及使用说明
