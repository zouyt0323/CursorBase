# 1. 元规则与最佳实践（Meta & Best Practices）

共 5 条，适用于：制定团队规则、评审项目、多技术栈/多智能体协作、统一代码生成标准。

| 序号 | 包名 | 说明 | 安装/来源 |
|-----|------|------|-----------|
| 1 | @prpm/creating-cursor-rules | 学习创建高质量 Cursor 规则的模板与最佳实践 | `prpm install @prpm/creating-cursor-rules` |
| 2 | @prpm/karen-repo-reviewer | AI 仓库评审，Karen 分数与过度工程/完成度/市场契合度 | `prpm install @prpm/karen-repo-reviewer` |
| 3 | @awesome-copilot/copilot-instructions | 高质量自定义指令指南（原 Copilot，原则适用于 Cursor） | PRPM / awesome-copilot |
| 4 | @ivangrynenko/code-generation-standards | 跨 PHP/JS/TS/Vue/JSX/TSX/Python 的通用代码生成标准 | PRPM |
| 5 | @ivangrynenko/multi-agent-coordination | 多智能体协作与工作流标准，适合复杂项目 | PRPM |

**安装示例（需先安装 prpm）：**
```bash
npm install -g prpm
prpm install @prpm/creating-cursor-rules
prpm install @ivangrynenko/code-generation-standards
```
