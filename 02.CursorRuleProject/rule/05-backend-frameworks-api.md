# 5. 后端框架与 API（Backend Frameworks & API）

共 6 条，适用于：Django/DRF、NestJS、GraphQL/Apollo 等 API 与全栈后端。

| 序号 | 包名 | 说明 | 安装/来源 |
|-----|------|------|-----------|
| 32 | @voltagent/django-developer | Django 最佳实践：ORM、模板与项目组织 | `prpm install @voltagent/django-developer` |
| 33 | @sanjeed5/django | Django 全栈：代码组织、迁移、部署与安全 | `prpm install @sanjeed5/django` |
| 34 | @sanjeed5/django-rest-framework | DRF：序列化、ViewSet、认证 | `prpm install @sanjeed5/django-rest-framework` |
| 35 | @sanjeed5/nestjs | NestJS 架构：依赖注入与模块划分 | `prpm install @sanjeed5/nestjs` |
| 36 | @sanjeed5/apollo-graphql | Apollo GraphQL：Schema、安全、缓存与 Resolver | `prpm install @sanjeed5/apollo-graphql` |
| 37 | @sanjeed5/graphql | GraphQL 通用规范：Schema、N+1、订阅 | `prpm install @sanjeed5/graphql` |
