# 示例子代理（Subagent）

本文件展示 `.cursor/agents` 下**项目级子代理**的写法与用途。

## 作用

子代理用于定义专用角色或流程，可与主 Agent 协作。项目级子代理（本目录）仅对当前仓库生效，适合团队共享。

## 可定义内容示例

- **角色**：如「前端规范检查员」「API 契约校验员」
- **输入**：接受哪些上下文（如当前文件、PR diff）
- **输出**：期望的产出（如检查清单、修改建议）
- **步骤**：执行顺序与注意事项

## 与用户级子代理的区别

| 位置 | 作用域 |
|------|--------|
| `.cursor/agents/*.md` | 当前项目，可版本控制 |
| `~/.cursor/agents/*.md` | 所有项目，个人使用 |

复制本文件并重命名即可新增子代理；在《Cursor文件夹说明.md》中可补充每个 .md 文件对应的功能说明。
